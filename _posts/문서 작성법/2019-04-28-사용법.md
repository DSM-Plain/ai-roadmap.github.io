jekyll-gitbook 테마는 다른 [Jekyll 테마][3]처럼 사용할 수 있으며, [Github Pages][1]에서 [원격 테마][2]를 지원합니다. [공식 가이드][4]도 참고하세요.

다음 방법 중 하나로 이 jekyll 테마를 자신의 사이트에 도입할 수 있습니다.

- 이 저장소를 [포크][5]하고 `_posts` 폴더에 마크다운 포스트를 추가한 다음, 자신의 Github 저장소에 푸시합니다.
- [`_config.yml`][6]에서 원격 테마로 사용합니다(이 사이트 자체에서 하는 것처럼).

```yaml
# Configurations
title:            Jekyll Gitbook
longtitle:        Jekyll Gitbook

remote_theme:     sighingnow/jekyll-gitbook
```

> ##### 팁
>
> 생성된 HTML 번들을 푸시할 필요가 없습니다.
