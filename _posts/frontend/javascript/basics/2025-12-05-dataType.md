# 데이터 타입(Data Type)

- 데이터 타입은 프로그래밍 언어에서 사용하는 데이터의 종류를 말한다.

## 자바스크립트의 데이터 타입 종류

### 원시 타입(primitive type)

- 원시 타입(primitive type): 원시 타입의 값은 변경 불가능한 값이며, 값으로 바로 전달한다.

- number, string, boolean, null, undefined, symbol이 원시 타입이다.

### 객체 타입(object type)

- 객체 타입(object type)은 객체 타입으로 원시 타입과 다르게 객체 타입은 다양한 데이터를 담을 수가 있다.
- 객체가 객체 타입이다.

## 원시 값(primitive type)

### 숫자(number)

자바스크립트는 정수와 실수를 구분하지 않고, 모든 수를 하나의 실수로 표현한다.

### 예시 코드

```javascript
const num = 10; //정수
const float = 10.123; //실수
```

### 문자열(string)

큰따옴표(") 또는 작은 따옴표(')로 둘러싸인 문자의 집합을 의미한다.

### 예시 코드

```javascript
const str = "javascript";
```

### 불리언(boolean)

- 참(`true`)과 거짓(`false`)을 표현하는 데이터 타입이다.
- 주로 조건문에 사용되며 0, null, false, NaN, undefined, 빈문자열은 `false`로 간주되고 그 외에 모든 값은 초기값이 `true`로 설정된다.

### 예시 코드

```javascript
const trueBoolean = true //참
const falseBoolean = false //거짓
```

### 심볼(symbol)

- ECMAScript 6부터 새롭게 생긴 추가된 타입이다.
- 유일하고 변경할 수 없는 타입으로 객체의 프로퍼티 식별자로 사용할 수 있다.
- 심볼 타입은 익스플로러에서 지원하지 않는다.

### 예시 코드

```javascript
const sym = Symbol("javascript"); // symbol 타입

const symObj = Object(sym); // object 타입
```

## null, undefined

### null

- null은 어떠한 변수의 값이 없다는 걸 명시하는 타입이다. 
- 참고로 자바스크립트는 대소문자를 구별하기 때문에 NULL, Null 등과 다르다. (이건 다른 데이터 타입도 똑같음)

### 예시 코드

```javascript
const Null = null; // null 값을 명시
```

### undefined

-undefined는 타입이 명시되지 않은 변수를 명시되는 타입이다. 즉, 변수 선언을 하였지만 값을 할당하지 않은 변수나 존재하지도 않은 변수를 가지고 올 때 `undefined`가 반환된다.

``` javascript
const num; // 값을 할당하지 않았으므로 nudefined 값을 반환함.

typeof secondnum; // 정의되지 않은 변수에 접근하면 undefined 값을 반환함.
```

## 객체 타입(object type)

### 객체(object)

- 자바스크립트의 기본 타입은 객체 그 자체이기도 하다.
- 객체는 실생활에서 인식할 수 있는 사물로 이해될 수 있으며, 이름(key)와 값(value)로 구성된 프로퍼티(property)의 정렬되지 않은 집합이다.

### 예시 코드

```javascript
const dog = {name: "해피", age: 3}; // 객체의 생성
```

### typeof 연산자

- `typeof`연산자는 피연산자의 타입을 나타내는 문자열을 반환하는 연산자이다.

### 예제

```javascript
typeof 10; /// number 타입

typeof "문자열"; // string 타입

typeof true; // boolean 타입

typeof undefined; // undefined 타입

typeof null; // object 타입
```
